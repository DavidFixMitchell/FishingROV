# Data Access And Attribution

This repository does not redistribute raw datasets, survey video, trained model
weights, or derivative datasets. Large shareable artifacts should live on an
artifact host such as Hugging Face, with clear licensing and attribution.

## Public Source Dataset

Primary source:

- Harlow, L., Ovchinnikova, K., & James, M. (2023). Data for Scallop
  (Pecten maximus) Identification in Natural Marine Habitats: A NetHarn Model
  Approach. Zenodo. https://doi.org/10.5281/zenodo.10156830

The Zenodo record is licensed CC-BY-4.0. Follow the source dataset's license and
attribution requirements when creating derivative datasets, trained weights, or
figures.

## Additional NatureScot Footage

Additional raw footage, metadata, or annotations should be requested from the
relevant data owners or custodians before use or redistribution.

Do not commit credentials, expiring links, private download URLs, or raw data to
this repository.

## Hugging Face Artifacts

Derivative datasets and trained weights based only on the public Zenodo dataset
can be shared publicly if they preserve CC-BY-4.0 attribution and clearly explain
the transformations applied. For outreach, it is better to be transparent that
these artifacts exist than to hide them.

Use private or access-controlled artifact repositories for anything that includes
non-public footage, unclear third-party terms, private transfer links, or partner
data that has not been explicitly cleared for redistribution.
